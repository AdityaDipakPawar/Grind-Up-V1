## ‚ú® Key Information

### Profile Completion Criteria
**College:** tpoName, collegeCity, avgCTC, placementPercent, placementRecordUrl
**Company:** recruiterName, recruiterEmail, companyBio, yearsOfExperience

### Key Endpoints
- `POST /auth/register/college` - College signup
- `POST /auth/register/company` - Company signup
- `POST /auth/login` - Login
- `GET /auth/me` - Get current user
- `GET /auth/check-profile-completion` - Check profile status

### Important Files
- Frontend Warning: ProfileCompletionWarning.jsx
- Backend Logic: authController.js
- Styling: ProfileCompletionWarning.css
- Auth Context: AuthContext.jsx

### User Flow
1. Signup ‚Üí Redirect to /profile
2. Complete profile ‚Üí Redirect to /home
3. Home page shows warning if incomplete
4. Click "Complete Profile" ‚Üí Back to /profile
